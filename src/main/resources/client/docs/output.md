Vetenskaplig output indelas i fem olika huvudkategorier:  

• Konstnärlig output  
• Publikationer  
• Konferensoutput  
• Immaterialrättslig output  
• Övrig output  

Indelningen i outputtyper ersätter tidigare publikationstyper.
