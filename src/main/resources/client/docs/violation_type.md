Här kan du göra uttag av poster med systemidentifierade brister i datakvaliteten utifrån SwePubs regler.
Bristerna delas in i feltyper av olika allvarlighetsgrader beroende på hur de påverkar olika typer av analyser. Posterna behöver åtgärdas i lokala system. Hantering av ofullständig metadata i finns beskriven i [Lathunden].
